# POS Billing App

## Overview
POS Billing App is a React Native application built using Expo for managing products, billing, receipts, and billing history.

## Features
- Product Management
- Add Products to Cart
- Generate Bills
- Receipt Preview
- Billing History
- Settings Management

## Tech Stack
- React Native
- Expo
- TypeScript
- Redux Toolkit
- AsyncStorage

## Installation

```bash
npm install
npx expo start
```
